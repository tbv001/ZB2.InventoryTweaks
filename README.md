# Inventory Tweaks

**Inventory Tweaks** is a mod for **Zumbi Blocks 2** that adds a few quality of life improvements to the inventory system, making inventory management easier and less tedious.

## Features

- Automatically sort items in the inventory with a button press or when opening the inventory.
- Shift-clicking an equippable item in the inventory will equip it or swap with an occupied equip slot. And shift-clicking an equipped item will unequip it.
- Coloring for the inventory slots based on the item tier in the slot.

## Installation

1. Make sure you have the latest version of **BepInEx 5** installed.
2. Download the latest DLL at the [releases](https://github.com/tbv001/ZB2.InventoryTweaks/releases) page.
3. Copy the DLL into the `plugins` folder of your BepInEx installation:

```
Zumbi Blocks 2 Open Alpha\BepInEx\plugins\InventoryTweaks.dll
```

## Configuration

On first launch with the mod installed, a configuration file is generated at:

```
Zumbi Blocks 2 Open Alpha\BepInEx\config\com.theblackvoid.inventorytweaks.cfg
```
